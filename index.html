DCB EVENTS
PHOTO VOTING PAYOUTS

A new way to run interactive, trust-based voting games with automatic crypto payouts — fully native to Discord. Server owners create photo-based voting challenges where participants earn rewards for correctly predicting the owner-selected winning photo.

01

Owner Creates an Event

Upload multiple images with a private winning nomination. Define prize pool and participant slots.

02

Participants Join & Vote

Members fill available seats and each submit one vote for the image they believe will win.

03

Event Starts Automatically

Once full, voting locks. Votes are tracked securely and transparently by DisCryptoBank.

04

Winners Get Paid Instantly

Anyone who selected the owner's nominated image wins. Rewards are auto-calculated and sent directly to connected wallets.

🎯

Owner-Controlled Outcomes

Private nomination ensures only the server owner knows the winning image. Provable fairness through recorded, transparent votes.

⚡

Instant Automated Payouts

No manual winner selection. No delays. No intermediaries. Prize pool splits automatically based on Event settings and pays directly from treasury.

🔗

Wallet-Verified Participation

Every participant's wallet is verified. Payments route securely from server treasury to individual connected wallets.

Getting Started

HOW DISCRYPTOBANK WORKS

A proof-based pipeline that scales across communities. Two wallets, smart routing, automated everything.

01

Connect Treasury

Admins use /wallet connect to link the server treasury wallet (one-time, immutable).

02

Set User Wallet

Users register their receiving wallet with /user-wallet — works across all servers.

03

Create Tasks or Events

Generate bulk paid microtasks across communities, or launch interactive DCB Events for engagement.

04

Submit Proof

Workers claim tasks, complete them, and submit screenshot + URL proof via Discord.

05

Auto Verify & Pay

DisCryptoBank verifies proof autonomously and routes SOL from treasury to the worker's wallet.

For Server Admins

Server Treasury Setup

Connect your server's treasury wallet. This is immutable for safety to prevent accidental wallet swaps.

/wallet connect <address>
✅ Independent treasury per server
✅ Prevents accidental wallet swaps
✅ Funds all payouts for that guild
For All Users

Personal Receiving Wallet

One wallet works across all servers running DisCryptoBank. Set once, earn everywhere.

/user-wallet
✅ Payments route treasury → your wallet
✅ Guild validation (only members get paid)
✅ Built for mass microtask payouts

COMMAND DICTIONARY

Everything DisCryptoBank can do — grouped by purpose.

🔐 Treasury & Wallets
/wallet connect <address>
Connect the server treasury (one-time, immutable).
/wallet balance
View treasury balance (SOL + USD).
/wallet info
Audit treasury configuration.
/user-wallet
Set your personal receiving wallet (works across servers).
📋 Bulk Tasks
/bulk-tasks create
Create a paid engagement campaign (title, description, payout).
/bulk-tasks list
List available tasks for members.
/bulk-tasks claim <task_id>
Self-assign a task to complete.
📸 Proof & Verification
/submit-proof <assignment_id>
Submit screenshot + exact URL via modal.
/approve-proof list
View pending proof submissions (role-restricted).
/approve-proof approve <proof_id>
Approve proof and trigger payout.
/approve-proof reject <proof_id>
Reject proof with a reason.
💸 Direct Payments
/pay <user> <amount> <currency>
Pay a user in SOL or USD (auto conversion). Smart routing from treasury → user wallet with guild validation.
Safety Controls
• Guild validation: only server members can be paid
• Smart routing: treasury → user wallet
• Immutable treasury: reduces admin mistakes

FREQUENTLY ASKED QUESTIONS

Is the server treasury changeable?

Treasury setup is designed as a one-time configuration for safety. This prevents accidental misroutes and ensures financial integrity.

Do users need a wallet on every server?

No. /user-wallet sets a personal receiving wallet that works across all servers running DisCryptoBank.

How do DCB Events determine winners?

The server owner privately nominates a winning image. Any participant who votes for that image is declared a winner and receives an automatic payout.

What is the DCB Event Manager?

A monthly subscription web dashboard that gives server owners a full UI for staff management, payroll scheduling, and transparent ledger audits.

Does DCB only work with Discord?

Discord is the primary platform today, but DisCryptoBank's payroll engine is built on APIs and Solana blockchain — designed to expand to Telegram, Slack, web apps, and more.

How do buyers see progress?

DisCryptoBank surfaces generated vs assigned vs pending completion — and approved proof screenshots can be displayed on IllySocial as live testimonials.
